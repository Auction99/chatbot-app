# ChatBot

免费开源的移动端私人 ChatGPT 应用, 支持 GPT3, GPT4 & Gemini Pro 模型。

> * Telegram频道: [ChatBot](https://t.me/chatbot_all)
>* Android端下载地址: [点我下载](https://github.com/ChatBot-All/chatbot-app/releases)
>* iOS端需自行编译

```shell
flutter pub get
cd ios
pod install
cd ..
flutter run
```
